Ist diese Webseite sicher?

    Teil 1

  • Ist die Verbindung zu dieser Webseite sicher?
  • Handelt es sich um eine Seite der DHBW Stuttgart?
  • Werden hier DSGVO-relevante Sachverhalte berücksichtigt?
  • Welche wichtigen Bestandteile fehlen auf dieser Seite (Vergleich mit anderen Web-Seiten)?
  • Welche Links / Unter-Seiten (im Menüband oben) gibt es in diesem Web-Auftritt?
  • Was ist github? Wem gehört github?
  • Und was verbirgt sich hinter github pages? Worin besteht der Vorteil?

    • Teil 2

  • Welche IP-Adresse hat diese Seite? Welcher Port?
  • Wer ist der Autor (engl. author) dieser Seite?
  • Können Sie versteckte (engl. hidden) html-Seiten finden?
  • Werden evtl. vertrauliche Informationen (z.B. passwords) vorgehalten?
  • Wozu kann der Web-Developer / Web-Entwickler (im FireFox) benutzt werden?
  • Sind vertrauliche Informationen auf dieser Webseite enthalten?
  • Was hat es mit der Datei robots.txt auf sich? Worin besteht eine Gefahr?
  • Das Scannen von Webseiten ist grundsätzlich möglich. Wie sieht es mit der Legalität aus?

    • Teil 3

  • Können Sie diese Datei öffnen? Darin finden Sie Details zu dieser Teil-Aufgabe.
  • Wie gehen Sie allgemein mit Links auf Dokumente (Word, Excel, Pdf, etc.) um?
  • Wozu dient die Seite Virustotal?
  • Was weiß Ihr Internet Browser über Sie? Nutzen Sie dazu z.B. ZENDAS oder myip.com!
  • Welchen Browser benutzen Sie (Firefox, Google Chrome, etc.)? Was sind Vor- bzw. Nachteile?
  • Welche Such-Maschine benutzen Sie? Haben Sie schon mal über Alternativen nachgedacht?
  • Wie sicher sind Sie im Internet unterwegs (Stichwort: Secure Browsing)?
